# Bodacare — Legal Documents for bodacare.com

Three standalone Korean-language legal documents for hosting on the marketing site.

## Files

- `privacy.md` — 개인정보처리방침 (Privacy Policy). Must be reachable at a stable public URL because App Store Connect requires it.
- `terms.md` — 이용약관 (Terms of Service).
- `sensitive.md` — 민감정보 수집 및 이용 동의 (Sensitive-info consent). Already required and shown in-app on signup; hosting publicly is optional but recommended.

## Suggested public URLs

- `https://bodacare.com/legal/privacy`
- `https://bodacare.com/legal/terms`
- `https://bodacare.com/legal/sensitive`

Use whichever paths your site framework prefers — the only hard requirement is that **the privacy URL must be public, stable, and listed in App Store Connect → App Privacy**.
